Пока проект можно поднять только на локальных серверах. 
Для этого: 
1. Клонируйте проект их гита
2. Откройте терминал в папке back, 
3. pip install -r requirements.txt - для установки всех необходимых зависимостей (чтобы наоборот сохранить в файл, используем команду pip freeze > requirements.txt)
4. Когда среда готова, выполните команду python main.py
5. Откройте терминал в папке front
6. Для установки всех необходимых зависимостей выполните npm install
7. для запуска фронтовой части выполните команду expo start
